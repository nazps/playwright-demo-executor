PreRequisities:
 - node V18.18.1
 - npm  9.8.1

Running:
- clone the repo: git clone https://github.com/nazps/playwright-demo-executor.git
In terminal: 

to only run tests:
```
npm install
npm run test
```
to start a server and execute tests via pressing a button:
```
npm start
```
navigate to: `http://localhost:3000`
press `Run Tests`
