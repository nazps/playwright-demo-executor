1. Installation:
   - git 
   - node V18.18.1
   - npm  9.8.1

3. Running:
- clone the repo:
- ```git clone https://github.com/nazps/playwright-demo-executor.git```

3. In terminal: 

- to only run tests:
```
npx playwright install
npm install
npm run test
```
- to start a server and execute tests via pressing a button:
```
npm start
```
navigate to: `http://localhost:3000`
press `Run Tests`
